### Система управления депозитами

##### T1 — Создание Rate Management Service API для расчета ставок

*Описание*:
Разработка микросервиса `Rate Management Service`, отвечающего за хранение базовых ставок, расчет персонализированных условий. Сервис будет предоставлять REST API для внутренних и внешних потребителей (кол-центры, CRM, веб-интерфейсы).

##### T3 - Создание Deposit Management Service API для управления депозитами

*Описание*:
Формирование сервиса, управляющего жизненным циклом депозита: открытие, изменение, закрытие. Реализация API для работы с депозитами через веб-интерфейс, кол-центр и CRM.

##### T4 — Деплой SFTP сервера

*Описание*:
Установка и настройка SFTP-сервера для безопасной передачи файлов с актуальными ставками партнерским кол-центрам. Обеспечение доступа с ограничением по правам и шифрованием.

##### T5 - Индивидуализация ставок

*Описание*:
Добавление логики расчета индивидуальных ставок на основе профиля клиента (история, баллы лояльности, сумма депозита и т.д.). Поддержка гибких политик и правил.

### АБС
##### T2 — Доработка функциональности: SMS-сообщения о статусах заявок

*Описание*:
Добавление механизма отправки SMS-уведомлений клиентам при изменении статуса заявки на депозит. Интеграция с внешним SMS-провайдером.

### Система кол-центра 

##### T6 - Интеграция системы управления депозитами с CRM

*Описание*:
Обеспечение двустороннего обмена данными между системой управления депозитами и CRM внутреннего кол-центра. Позволяет операторам видеть актуальные ставки и создавать заявки.
